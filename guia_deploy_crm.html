1
Baixe o arquivo do CRM já pronto
Clique no botão abaixo para baixar o arquivo index.html que acabei de gerar.
Salve em qualquer pasta do seu computador
2
Crie uma conta no GitHub github.com
O GitHub guarda o código do seu site. É gratuito.
Acesse github.com e clique em Sign up
Crie conta com seu e-mail
Depois clique em New repository (botão verde)
Nome do repositório: crm-loja → marque Public → clique Create repository
Na próxima tela clique em uploading an existing file → arraste o index.html → clique Commit changes
3
Publique no Vercel vercel.com
O Vercel pega seu código do GitHub e coloca online grátis em segundos.
Acesse vercel.com → clique Sign up → entre com a conta do GitHub
Clique em Add New → Project
Selecione o repositório crm-loja
Clique Deploy — não precisa mexer em nada
Pronto! Vai gerar um link tipo crm-loja.vercel.app
4
Como atualizar depois
Sempre que evoluirmos o CRM aqui, você só faz isso:
Baixa o novo index.html que eu gerei
Entra no GitHub → repositório crm-loja
Clica no arquivo index.html → ícone de lápis (editar) → aba Upload
Sobe o arquivo novo → Commit changes
O Vercel detecta automaticamente e publica em ~30 segundos
Tudo gratuito. GitHub + Vercel têm planos free que cobrem 100% do que você precisa. Os dados ficam salvos no navegador de cada loja — se quiser sincronizar entre lojas em tempo real, a gente adiciona banco de dados depois.
